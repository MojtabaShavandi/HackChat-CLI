## Features

- Real-time chat in your terminal
- Colorful messages for better readability
- View online users
- Private messages (`?pm user message`)
- Useful commands: `?help`, `?me`
- Lightweight and works on Linux, macOS, and Windows

## Requirements

- Python 3.8+
- `websockets`
- `prompt_toolkit`
- `colorama` (for Windows)

Install dependencies using pip:

```bash
pip install -r requirements.txt
Usage
Clone the repository:


git clone https://github.com/YourUsername/SCORPION-HackChat-CLI.git
cd SCORPION-HackChat-CLI
Run the client:

bash
python3 HackChatCli.py
Enter your room name and username, then start chatting!

Commands
Command	Description
?help	Show available commands
?me	Show your username
?pm user msg	Send a private message to a specific user

Notes
Works best in terminals that support ANSI colors.

For Windows users, colorama is included to enable colors.

Keep your terminal wide enough to display messages clearly.
